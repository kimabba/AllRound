# 스토어 그래픽 에셋

## 준비 완료

- `app-icon-512.png` — Google Play 앱 아이콘, 512×512
- `feature-graphic-1024x500.jpg` — Google Play 피처 그래픽, 1024×500

앱 내부 마크의 브랜드 색상과 형태를 사용했습니다. 편집 가능한 원본은 아래 파일입니다.

- `app/assets/brand/app_icon.svg`
- `app/assets/brand/play_feature_graphic.svg`

iOS AppIcon, Android mipmap, Web icon도 같은 원본으로 교체했습니다. iOS 아이콘 전 크기는 알파 채널이 없음을 검증했습니다.

## 남은 에셋

- Google Play 폰 스크린샷 최소 2장(권장 5장)
- App Store 6.7/6.9인치 스크린샷 세트
- iPad 지원을 유지할 경우 iPad 스크린샷 세트

스크린샷 시나리오는 `docs/store-listing.md`의 5장 구성을 따르고 스피드건 화면은 포함하지 않습니다.
