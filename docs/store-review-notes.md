# 스토어 심사 노트 — 올라운드 1.0.0

> App Store Connect와 Google Play Console에 복사할 공통 초안입니다.
> 비밀번호는 저장소에 커밋하지 않습니다. 제출 직전에 콘솔에만 입력합니다.

## 심사용 계정

- 일반 사용자 이메일: `[콘솔에 입력]`
- 일반 사용자 비밀번호: `[콘솔에만 입력]`
- 관리자 이메일: `[콘솔에 입력 — 웹 관리자 신고함 검토가 필요한 경우]`
- 관리자 비밀번호: `[콘솔에만 입력]`

계정은 이메일 인증이 완료되어 있으며 종목, 등급, 지역 설정과 테스트용 클럽 가입을 미리 준비합니다. 제출 직전 두 계정으로 로그인을 다시 확인합니다.

## 로그인

- 이메일/비밀번호 또는 Google 로그인을 지원합니다.
- 심사는 위 이메일 계정을 사용하면 별도의 Google 계정 연결 없이 진행할 수 있습니다.
- 자체 이메일 로그인을 제공하고 로그인 과정에서 서비스 제공에 필요한 정보만 요청하므로 Sign in with Apple은 이번 버전에 포함하지 않았습니다.

## 핵심 기능 확인 경로

1. **AI 코치**: 첫 번째 탭에서 대회·규정·구장 질문을 입력합니다. 답변은 앱에 등록된 데이터에 기반하며 웹을 임의 검색하지 않습니다.
2. **대회**: 대회 탭에서 지역·등급·접수 상태 필터와 월별 캘린더를 확인합니다.
3. **클럽**: 클럽 탭에서 가입한 테스트 클럽을 열어 게시글·댓글·모임을 확인합니다.
4. **회원 탈퇴**: 더보기 → MY → 계정 → 회원 탈퇴.

## AI 및 UGC 안전 기능

- AI 답변에는 AI 생성 안내가 표시되며 각 답변의 **AI 답변 신고** 버튼으로 신고할 수 있습니다.
- 클럽·게시글·댓글·모임의 더보기 메뉴에서 신고 또는 작성자 차단을 선택할 수 있습니다.
- 더보기 → 차단 관리에서 차단한 사용자를 확인하고 해제할 수 있습니다.
- 커뮤니티 콘텐츠를 처음 작성하거나 클럽을 생성·가입할 때 커뮤니티 이용약관 필수 동의 화면이 표시됩니다.
- 신고는 관리자 웹 화면의 **신고 · 제재** 메뉴에서 검토하며 콘텐츠 삭제와 기간별 제재를 지원합니다.

## 결제·광고·추적

- 앱은 무료이며 인앱 결제와 구독이 없습니다.
- 광고 SDK와 사용자 추적 SDK를 사용하지 않습니다.
- GPS 위치정보를 수집하지 않습니다. 지역은 사용자가 직접 선택하는 설정값입니다.

## 제출 직전 입력·확인

- [ ] 일반 사용자 데모 계정 이메일/비밀번호를 콘솔에 입력
