# 스토어 등록 메타데이터 (올라운드 / AllRound)

> 대상: 테니스·풋살 동호인 통합 앱. **스피드건은 MVP 스코프 아웃이라 리스팅에 포함하지 않는다.**
> AI 답변은 웹 검색이 아니라 **앱에 등록된 대회·규정·구장 데이터 기반**이므로 그렇게 표기(심사 정확성).

## Google Play Store

### 앱 이름 (30자 이내)
올라운드 — 테니스·풋살 동호인

### 짧은 설명 (80자 이내)
AI에게 물어보는 대회 일정, 클럽 커뮤니티, 규정·구장 정보까지 한 곳에서.

### 전체 설명
올라운드는 테니스·풋살 동호인을 위한 통합 정보 앱입니다. 대회 일정을 AI로 검색하고, 클럽에서 동료와 소통하며, 규정과 구장 정보까지 한 앱에서 확인할 수 있습니다.

**AI 코치 — 자연어로 물어보세요**
"이번 달 우리 지역 대회 있어?", "타이브레이크 규칙 알려줘" 처럼 대화하듯 물어보면, 앱에 등록된 대회·규정·구장 정보를 바탕으로 라운드 코치가 답변합니다. 답변은 실시간 스트리밍으로 표시되고, 출처를 함께 안내합니다.

**대회 — 전국 동호인 대회 검색·캘린더**
각 협회에서 수집한 테니스·풋살 대회 정보를 지역·등급·접수 상태(접수중/마감/종료)로 필터링해 확인할 수 있습니다. 캘린더로 월별 일정을 보고, 관심 대회를 즐겨찾기하면 접수 마감 전에 알림을 받습니다. 빠진 대회는 직접 제보할 수 있습니다.

**클럽 — 우리 동호회 커뮤니티**
클럽을 만들거나 가입해 게시판으로 소통하고, 모임 일정을 만들어 참석 여부를 모읍니다. 멤버·운영진·클럽 소개까지 한 곳에서 관리합니다.

**규정·구장**
테니스·풋살 규정을 검색하고, 내 주변 구장을 찾아볼 수 있습니다.

**주요 특징**
- 테니스·풋살 두 종목 지원 (종목 전환)
- 등록 데이터 기반 AI 코치 (실시간 스트리밍 답변, 출처 표시)
- 지역·등급·상태별 대회 검색 + 월별 캘린더
- 즐겨찾기 대회 마감 알림
- 클럽 커뮤니티(게시판·모임·참석·멤버)
- 규정 검색 · 구장 찾기

## 공통

### 앱 아이콘 설명 (디자이너 참고용 — 백과장)
- 형태: 테니스·풋살 **두 종목을 아우르는** 심볼(특정 공 하나에 치우치지 않게). "올라운드"의 만능/멀티스포츠 뉘앙스.
- 색상: 브랜드 컬러 기준(디자인 시스템 확정값 사용).
- 스타일: 플랫, 작은 크기에서도 식별되는 단순한 구성. 텍스트 삽입 금지.
- 금지: 스피드건/라켓 등 특정 기능·복잡 요소 강조 배제.

### 스크린샷 시나리오 (5장 — 백과장)
> 스피드건 제외. 실제 MVP 화면 기준.

1. **AI 코치(메인)** — "이번 주 우리 지역 대회 있어?" 질문에 대회 카드가 스트리밍으로 답변되는 장면. 캡션: "AI에게 물어보세요 — 대회·규정·구장까지"
2. **대회 목록/캘린더** — 지역·상태 필터가 적용된 대회 카드 목록 또는 월별 캘린더. 캡션: "지역·등급·상태로 대회를 찾아보세요"
3. **클럽 커뮤니티** — 클럽 상세(게시판·모임 일정·멤버) 화면. 캡션: "우리 동호회, 한 곳에서"
4. **규정/룰북** — 규정 검색 결과 또는 상세. 캡션: "궁금한 규칙, 바로 확인"
5. **종목 전환/프로필** — 테니스 ↔ 풋살 전환과 내 프로필. 캡션: "테니스도 풋살도, 올라운드 하나로"
