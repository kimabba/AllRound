# 한국 동호인 테니스 등급 체계 리서치 보고서

생성: 2026-05-09
대상: Match-up 앱 데이터 모델 결정 (rule_articles 콘텐츠 + user_sports 등급 모델)
범위: 전국 단위 4개 단체(KTA, KATO, KATA, KTFS) + 17개 시·도 협회

---

## 0. 핵심 결론 (Executive Summary)

### 가장 중요한 발견 5가지

1. **단일 enum으로 매핑 불가능.** 한국에는 사실상 4개의 전국 단위 동호인 단체(KTA, KATO, KATA, KTFS)가 있고 각각 등급/부서/점수 체계가 다름. 2025년 1월 KTA 초대 랭킹위원장도 다음 기사에서 "3개 랭킹은 난센스"라며 통합을 제안했음. **Match-up 현재 enum (`rookie/div5~div1`)은 어느 단체와도 1:1 일치하지 않음.**

2. **부서명은 사실상 표준화됨.** 단체마다 점수·승급 규칙은 다르지만 부서명은 거의 공통:
   - 남자: **마스터즈 / 챌린져(베테랑) / 지도자 / 위너스(오픈) / 금배 → 은배 → 동배 / 신인**
   - 여자: **오픈 / 국화 / 개나리 / 신인(비입상자)**
   - 페어 합산 점수 (1.0~10.0) 모델은 KTA가 표준화

3. **광역시-도 통합 권역 패턴 강함.**
   - **광주 ↔ 전남**: 사실상 단일 권역. 합동 상벌위원회까지 운영. 광주협회 랭킹규정을 양 지역이 공유.
   - **대구 ↔ 경북**: 아카시아배 등 합동 대회 다수.
   - **부산 ↔ 울산 ↔ 경남**: 영남권 공유 추정.
   - 데이터 모델에 "권역" 개념 필요할 가능성.

4. **점수제 vs 등급제 혼재.**
   - 전북: **명시적 점수제 전환** (남자 합산 2~13점, 여자 2~12점)
   - 제주: 개인등록 기준표 매년 갱신
   - 그 외: 클럽 단위 등급 + 부수 출전
   - 단순 enum보다 `(부서, 점수, 자기신고 등급)` 다중 필드 필요

5. **승급 후 시간의존적 강등.** KTA 2024 규정: "입상 승급 후 만 1년 경과 시 본래 등급 복귀, 추가 입상 시 1년 연장". 즉 등급은 시간 함수 → DB에 `expires_at` 같은 만료 필드 필요할 수 있음.

### 데이터 모델 권고 (예비)

현재 `user_sports.grade text` (단일 enum) → 다음으로 변경 검토:

```
user_sports
  user_id uuid
  sport sport
  ── 한국 테니스 특화 (sport='tennis'일 때)
  tennis_org tennis_org_enum NULL          -- 'kta' | 'kato' | 'kata' | 'ktfs' | 'local'
  tennis_division text NULL                -- '오픈부' / '국화부' / '베테랑부' 등 부서명
  tennis_score numeric(3,1) NULL           -- 1.0~10.0 (KTA 합산 점수제)
  tennis_self_grade text NULL              -- '5부' / '신인' 등 자기신고 (자유 입력)
  tennis_grade_expires_at date NULL        -- 시간의존 등급 만료
  ── 풋살은 단순 유지
  futsal_grade text NULL                   -- 'beginner|intermediate|advanced'
  is_primary boolean
```

별도 `tennis_division_dictionary` 테이블로 (org × division × 표시명 × 동의어) 매핑 운영 권장.

## 1. 전국 협회

### 1.1 대한테니스협회 (KTA, Korea Tennis Association)

- **URL**: https://www.kortennis.or.kr/ · https://join.kortennis.or.kr/
- **위상**: 대한체육회 정식 가맹 단체. 엘리트(선수) + 생활체육 통합 관리. 17개 시·도 협회의 상위 조직.
- **등급 체계 (2024.02.13 개정 「생활체육 복식 랭킹관리지침」)**:
  - **9개 부서**: 전국신인부 / 남자 오픈부 / 세미 오픈부 / 베테랑부 / 지도자부 / 개나리부(여자) / 국화부(여자) / 혼합복식부 / 청·장년부
  - **점수 체계**: 페어 합산 점수제 (1.0~10.0)
    - 오픈부: 페어 합산 11.0 이하
    - 세미오픈부: 1.0~7.0, 합산 8.0 이하
    - 베테랑부: 1.0~8.0, 합산 9.0 이하
    - 국화부: 1.0~9.0, 합산 10.0 이하
- **승급 기준**:
  - 입상 승급 후 만 1년 경과 시 본래 등급 복귀
  - 추가 입상 시 그 시점부터 만 1년 연장
  - 만 60세 이상 연령 해제자: 입상 후 즉시 본래 등급 환원 + 매년 1월 1일 한 단계 하향
  - 페어 합산 우승 횟수에 따라 최소 페어 연령 합산 조건 (105세, 107세 등)
- **가입**: KTA 통합시스템(join.kortennis.or.kr) 회원가입 + 클럽·시·도협회 등록 연동
- **주요 대회**: 생활체육대회(전국), 16개 시·도지부 연계 대회, 그랜드슬램
- **호환성**: KTA-KATO 우승은 일부 인정. KATA 우승은 2023.01.01 이후 인정 안 됨.
- **활성**: ✅ 정상 (2024·2026 규정 갱신 확인)

### 1.2 (사)한국테니스발전협의회 (KATO, Korea Amateur Tennis Organization)

- **URL**: https://www.kato.kr/ · 규정: https://www.kato.kr/katoRules
- **등급 체계**: **그룹(대회 권위) × 부서(출전 자격)** 의 2축 구조
  - **그룹 (6단계)**: 마스터즈(MA) / A그룹 / 1그룹 / 2그룹 / 3그룹 / 4그룹(지역대회)
  - **부서**: 마스터즈부 / 챌린져부 / 베테랑부 / 지도자부 / 위너스부 / 국화부 / 개나리부
  - **부서별 자격**:
    - 마스터즈부: 만 55세 이상 우승자
    - 챌린져부: 마스터즈/베테랑 1회 우승자(연도해제 없음)
    - 베테랑부: 만 55세 이상
    - 지도자부: 만 40세 이상 + 지도자 자격
    - 위너스부: 페어 중 1명 이상 우승자
    - 국화부: 만 40세 이상 (혼합)
    - 개나리부: 비우승자만 (우승 시 진급)
- **승급**: 그룹 요건 충족 + KATO 정기총회 최종심의 통과. 한 번에 두 단계 이상 승격 불가.
- **가입**:
  - 단체회원 연회비: MA그룹 70만원 / A·1그룹 60만원 / 2·3그룹 50만원 / 4그룹 30만원
  - 개인회원: 홈페이지 회원가입 또는 랭킹대회 참가신청만으로 가입
- **호환성**: KATO·KATA·KTFS 모두 ATP/WTA처럼 1년 단위 랭킹 갱신. **공식 환산표 없음**. "단체별 신인 우승자는 마스터즈/베테랑부에서 비우승자로 간주" 조항이 있어 외부 단체 우승 이력 일부 인정.
- **활성**: ✅ 정상

### 1.3 (사)한국테니스진흥협회 (KATA, 구 한국동호인테니스협회)

- **URL**: http://new.ikata.org/ · http://m.ikata.org/ (모바일)
- **등급 체계**: 부서 중심
  - 오픈부 / 마스터즈부 (상위)
  - 신인부 / 챌린저부 (하위)
  - 베테랑부
  - 국화부 / 개나리부 (여성)
  - 페어 합산 점수 방식
- **승급**:
  - **베스트 15 방식**: 32강 이상 진출한 KATA 공인 대회 중 상위 15개 성적의 합으로 랭킹 산정
  - 1년 단위 랭킹 갱신 (점수 방어 실패 시 하락)
- **연혁**: 1995년 KTA 동호인위원회 → 2007년 한국동호인테니스협회 → 한국테니스진흥협회 개명
- **주요 대회**: 하나은행컵 KATA 투어 (전국, 수도권 비중 높음)
- **활성**: ⚠️ **사이트 SSL 인증서 호스트명 오류** (`ERR_TLS_CERT_ALTNAME_INVALID`). Flutter in-app webview에서 KATA 페이지 임베드 시 SSL 경고 발생 가능 — **앱 운영상 주의**
- **호환성**: KATO와 부서명은 거의 동일하나 점수·등급 체계 별개. 수도권 동호인 비중 높음.

### 1.4 국민생활체육전국테니스연합회 (KTFS, "생체")

- **URL**: http://www.kgtfs.com/ (timeout 빈발)
- **등급 체계**: 자체 경기규칙 + 랭킹규정. 시·도연합회 통합 구조. **본문 직접 확인 어려움 — 추후 수기 보완 필요**
- **메모**: kgtfs.com은 실제로는 **경기도 테니스 연합회** 도메인. 전국 단위 KTFS의 별도 통합 사이트가 없을 가능성. 대한체육회 산하 시·도연합회가 분산형으로 운영.
- **주요 대회**: 대한체육회장기 생활체육 전국테니스대회, 시·도연합회장배 전국 동호인대회
- **활성**: ⚠️ 사이트 응답 불안정

### 1.5 한국시니어테니스연맹 (KSTF) — 참고용

- **URL**: https://kstf.kr/ · https://www.kstftennis.com/?pg_idx=204
- **위상**: 시니어(만 50세+) 특화. 본 리서치 범위 외이지만 향후 시니어 사용자 대응 시 별도 매핑 필요
- **활성**: ✅ 정상

## 2. 수도권

### 2.1 서울 — 서울특별시테니스협회

- **URL**: https://seoultennis.com/ · 서울시체육회 산하 페이지: https://www.seoulsports.or.kr/tennis
- **인스타**: @seoul_tennis
- **등급 체계**: **공식 사이트에 등급/부수 명문화 안 됨**. KTA 시·도지부로서 KTA 표준 부서 따르는 것으로 추정. 자체 정규경기는 KTA·KATO·KATA 부서명 그대로 사용.
- **회원가입**: 사이트에 가입 페이지 없음 — 클럽 단위 등록을 통해 KTA 통합시스템 연동 구조
- **연락처**: 서울 중랑구 망우로 182 서울시체육회관 405호 / 02-490-2790 / tennis2790@naver.com
- **활성**: ✅ (2026.05 기준 뉴스·공지 갱신 활발)

### 2.2 경기 — 경기도테니스협회 / 경기도 테니스 연합회

- **URL**:
  - **연합회**(생활체육): http://www.kgtfs.com/
  - **협회**(KTA 산하): 별도 도메인 미확인 — KTA 통합시스템에서 시·도지부 형태
- **등급 체계**: 시·군 협회마다 자체 등급. 경기도 단위 통일 등급 체계 명문화 미확인.
  - 예: **성남시테니스협회** — 베테랑부/금배부/은배부/동배부 4단계 + 입상 2회 시 승급
- **시·군 협회 활성**: 성남(✅), 광명(✅), 고양(✅), 과천(✅), 김포, 의정부, 도봉구 등 다수
- **주요 대회**: 의정부시협회장배 전국 대회(38회), 부천복사배 전국 대회 등 시·군 단위 전국 대회
- **활성**: ⚠️ 도메인 통합성 약함

### 2.3 인천 — 인천광역시테니스협회

- **URL**: **공식 사이트 없음**. 인천광역시체육회(https://www.icsports.or.kr/) 산하 단체로 등록만 확인.
- **주소**: 인천광역시 부평구 열우물로 164, 2층 214호 (열우물경기장 내)
- **등급 체계**: **정보 부족 — 추후 수기 보완 필요**. 산하 시·군·구 협회(인천 서구 테니스협회 등)에서 KTA·KATO 부서명 사용
- **주요 대회**: 남동오픈 전국 동호인 테니스대회(KATO 인증), ITF 인천국제여자테니스투어(엘리트)
- **활성**: ⚠️ **공식 웹사이트 부재** — 카페·체육회 페이지·언론 보도로만 활동 확인

## 3. 광역시 (5개)

### 3.1 부산 — 부산광역시테니스협회 (BSTA)

- **URL**: http://www.bsta.kr/ (SSL 오류로 fetch 실패)
- **등급 체계**: KTA 표준 부서 + 합산 점수제 추정
- **주요 대회**:
  - 부산광역시장배 슈퍼컵 (BSSC, http://www.bssc.or.kr/)
  - 부산광역시테니스협회장배
- **호환성**: KTA-KATO 상호 인정. KATA 우승 인정 안 됨.
- **활성**: ✅

### 3.2 대구 — 대구광역시테니스협회 (DTA)

- **URL**: http://tgtennis.co.kr/ (별도 도메인 http://www.tgtennis.net)
- **위치**: 대구 달서구 두류공원로 161 / 053-357-1679
- **구성**: 산하 7개 구·군 협회 + 여성/시니어 전문위원회
- **등급 체계**: KTA 표준 9 부서 + 합산 점수제 추정 (직접 확인 실패)
- **주요 대회**:
  - 대구광역시테니스협회장배 (KTA 랭킹대회, "POWERFUL DAEGU" 18회)
  - 대구광역시테니스연합회장배
  - 구청장배·쉬메릭배·동호인 초보자 대회
- **활성**: ✅

### 3.3 광주 — 광주광역시테니스협회 (GJTA) ★ Match-up 시드 지역

- **공식 조직**: 1986년 설립 (구 광주직할시 테니스협회). 약 130개 클럽, 동호인 약 15,000명
- **URL**:
  - http://www.gjtennis.net/ (구 사이트, 게시판/규정)
  - https://gjtennis.kr/ (신 사이트, 생활체육부)
- **위치**: 광주 남구 진월동 (진월국제테니스장 인근) / 062-714-3244
- **등급 체계 — 광주 자체 부서**:

  | 부서 | 비고 |
  |------|------|
  | **골드부** | 최상위 (오픈부 우승자급 / 마스터즈 성격) |
  | **남자일반부** | 청·장년부 통합 일반 부서 |
  | **여자금배부** | KTA 국화부에 해당 |
  | **여자신인부** | KTA 개나리부에 해당 |
  | **국화부** | 여자 베테랑급 |
  | **지도자부** | 선수출신·동호인 지도자 |

- **광주 vs KTA 표준 매핑 (추정)**:

## 5. 종합 분석

### 5.1 등급 체계의 다층성

| 계층 | 단위 | 표시 |
|------|------|------|
| 전국 | KTA / KATO / KATA / KTFS | 부서 + 점수 (1.0~10.0) |
| 광역 | 광역시·도협회 | KTA 표준 차용 ~ 자체 부서 (광주 골드부, 전북 점수제 등) |
| 시·군 | 시·군·구 협회 | 자체 등급 (성남 동/은/금배부 4단계 등) |
| 클럽 | 동호회 | 자체 등급 (클럽 1부, 2부 등) |

**현재 Match-up enum (rookie/div5~div1)은 어느 계층과도 1:1 일치 안 됨.**

### 5.2 통합 권역 매핑 (대회 자동 매칭에 활용 가능)

| 권역 | 구성 | 호환성 |

| 영남 동남권 | 부산 + 울산 + 경남? | ★★★★ (부산광역시장배에 울산 다수) |

### 5.3 데이터 모델 권고 (3가지 옵션)

#### 옵션 A. 현재 enum 유지 + rule_articles 텍스트 보강 (가장 빠름)

- `user_sports.grade`는 그대로 (rookie/div5~div1, 6단계)
- `rule_articles`에 협회별 등급 체계 글 추가 → 챗봇 RAG가 답변
- 변경 0건. 콘텐츠만 추가.
- **단점**: 정형 매핑 불가. 사용자가 "내가 광주 골드부인데 어떤 KATO 그룹과 동등?" 같은 질의를 자동 매칭으로 답할 수 없음.
- **장점**: 모델 변경 없음, 1~2일 내 구현 가능.

#### 옵션 B. tennis_division enum + 점수 컬럼 (정형 모델)

```sql
create type tennis_org as enum ('kta','kato','kata','ktfs','local');
create type tennis_division as enum (
  'masters', 'challenger', 'veteran', 'coach',
  'winners', 'open', 'semi_open',
  'gold', 'silver', 'bronze',
  'rookie', 'national_rookie',
  'gukhwa',  -- 국화 (여자 상위)
  'gaenari', -- 개나리 (여자 신인)
  'mixed_doubles'
);

alter table user_sports add column tennis_org tennis_org;
alter table user_sports add column tennis_division tennis_division;
alter table user_sports add column tennis_score numeric(3,1);
alter table user_sports add column tennis_grade_expires_at date;
```

- 별도 `tennis_division_dictionary` 테이블에 (org × division × 표시명 × 동의어 × 점수 범위) 매핑.
- 광주 자체 명칭 (골드부, 금배부 등)은 dictionary에서 표시명 매핑.
- **장점**: 정형 매핑 가능. 권역별 자동 필터링 강화.
- **단점**: 마이그레이션 + UI(온보딩) + RPC 변경 큼. 사용자가 자기 등급을 모르면 입력 어려움.

#### 옵션 C. 자기신고 자유 텍스트 + 점수 (실용적 절충)

```sql
alter table user_sports add column tennis_org_note text;        -- "KATO 챌린져부", "광주 골드부", "성남 은배" 등 자유
alter table user_sports add column tennis_score numeric(3,1);   -- 합산 점수 (없으면 NULL)
```

- enum은 그대로 유지 (rookie/div5~div1, 자동 필터링용)
- 추가로 자유 텍스트 + 점수 입력 (선택)
- 챗봇이 자유 텍스트를 RAG 컨텍스트로 활용
- **장점**: 모델 변경 작음. 사용자가 자기가 아는 표현으로 입력 가능.
- **단점**: 정형 매핑 약함. 정확한 권역 매칭은 챗봇 답변 품질에 의존.

## 7. 운영적 시사점

1. **사이트 안정성 — 크롤러·임베드 운영 시 주의**:
   - KATA: SSL 인증서 호스트명 오류
   - KTFS: timeout 빈발
   - 광주·전남: ECONNREFUSED 잦음
   - 부산·울산: SSL 또는 timeout
   - **앱 내 webview 임베드보다 외부 브라우저 열기로 처리 권장**

2. **신뢰할 수 있는 단일 출처**: KTA 통합시스템(join.kortennis.or.kr)의 PDF가 가장 권위 + fetch 가능. **이 단일 소스를 캐시·정형화해서 앱 내 표시 권장**.

3. **라벨 정책**: KTA 영문 약자가 미국 재미한인 KATA(Korean American Tennis Association)와 충돌. UI에서는 한글 명칭 우선 + 영문 약자는 보조.

4. **광주 메이저 시드 처리**: 등급명이 아닌 클럽 고유명. clubs 테이블의 name 컬럼은 그대로 유지하고, 등급으로 해석하지 않도록 코드 검증 필요.

5. **시간의존 등급**: KTA 1년 유예 규정 → 사용자 등급에 만료일 또는 마지막 갱신일 추가 검토.
