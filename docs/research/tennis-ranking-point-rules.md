## 3. 광주 규정 상세 (현행 2026-03-17 시행)

### 3.1 배점표 — 제12조 그레이드별 랭킹 포인트

> ◇ 우승자 랭킹 포인트 ◇ (지도자부, 골드부, 여자 국화부, 여자 금배부)
> ◇ 비 우승자 랭킹 포인트 ◇ (남자 일반부, 여자 신인부)

### 3.2 대회 등급(그레이드) = 대회 그룹 — 제11조

포인트 그레이드 A/1/2/3은 선수 등급이 아니라 **대회가 몇 개 부서를 여느냐**로 결정된다.

| 대회 그룹 | 경기 부수 | 요건 (원문 요지) |
|---|---|---|
| A | 5 | 지정 경기부 + 선택 경기부 중 5개 부 이상 |
| 1 | 4 | 지정 경기부 + 선택 경기부 중 4개 부, 또는 남자일반부·골드부·여자부 |
| 2 | 3 | 지정 경기부 3개 부 |
| 3 | 2 | 지정 경기부 2개 부 |

원문 인용 (그룹 조정·특례):

> 라. 광주협회에서 주최, 주관하는 광주광역시장배, 광주협회장배는 A그룹 여성 위원회에서 주최, 주관하는 대회는 2그룹 대회로 한다.
> 마. 남자 일반부, 여자부, 지도자+골드부, 골드부로 개최한 대회는 기본부수에 1개 부서를 추가하여 등급에 산정한다.
> 사. 광주, 전남에서 개최되는 KTA 전국 동호인 대회와 생활체육 랭킹대회와 중복 시 협의하여 부서를 조정할 수 있으며 …

**전국대회(KTA) 취급** — 제13조 18항:

> 광주, 전남에서 개최되는 KTA 전국 동호인 대회는 광주 생활체육 랭킹규정에 준하여 회원단체 대회로 인정하며 **3그룹 대회에 준하여 배점**한다. 단, 포인트 배점은 KTA 규정(제14조(랭킹 포인트 및 배점)의 그레이드에 준하며 동일 대회에서 2개부 이상 참가하여 얻은 포인트는 참가부서 중 높은 포인트를 해당 소속 부서에 배점한다.

**시·도 대표 선발전** — 제12조 1항:

> 대한테니스협회에서 주최, 주관하는 대통령기, 대한체육회장기, 대축전에 시. 도를 대표하여 참가한 선수에게는 … 대회 입상 성적에 따라 **2그룹 포인트**를 배점하고 참가점수는 **2그룹의 8강 포인트**를 배점한다.

### 3.3 집계 방식 — 베스트 N (⚠️ 문서 내부 모순)

같은 규정 안에서 두 값이 나온다. **자동 계산 구현 전 협회 확인 필수.**

> **제13조 4항.** 랭킹순위는 그룹 및 그레이드에 의해 생활체육 랭킹대회에 출전하여 얻은 포인트 및 시. 도 대표선수 출전 점수를 포함한 **"베스트 25제"** 를 채택한다.

> **제27조. 종합시상.** 생활체육 회원단체가 주최 주관하는 대회에 참가하여 받은 대회별 포인트를 **"베스트 15제"** 를 적용하여 시상함을 원칙으로 하며 …

정리: **랭킹 순위 = 베스트 25 / 연말 종합시상 = 베스트 15** 로 읽히지만, 규정이 이를 명시적으로 구분하지 않는다.

### 3.4 배점 제외 조건 — 제13조

| 조항 | 내용 |
|---|---|
| 13-1 | 배점 부서: 지도자부 · 남자오픈부 · 골드부 · 남자일반부 · 여자 우승자부(국화·금배) · 여자신인부. **그 외 부서는 배점 없음** |
| 13-5-5) | 베테랑부 · 부부부 · 비입상자부 · 초급자부는 **배점 없음** |
| 13-6-8) | 대회 등급 산정의 기준이 되는 부서는 배점하지 않는다 |
| 13-10 | **참가팀수 32개팀 미만인 부서는 랭킹 포인트를 배점하지 않는다.** 단, 지도자부는 참가팀 수 관계없이 배점 |
| 13-13 | 협의 없이 순연한 대회는 배점하지 않고 페널티 |
| 13-14 | 고의적 우승 회피 시 준우승 상금·포인트 적용 |
| 13-15 | 규정 위반으로 얻은 포인트는 사후에라도 박탈 |
| 13-20 | 랭킹 업데이트는 대회 폐막 후 3일 이내 홈페이지 반영 |

### 3.5 이중 적립 규칙 — 제13조 2항 (정형화 난이도 높음)

> 1) 남자는 생활체육 랭킹대회 오픈부, 남자 일반부에서 우승하였을 경우 우승 당해 년은 남자 일반부 소속으로 하여 배점하고 우승 이후부터는 골드 부에 참가하여야 하며 일반부와 골드 부에서 얻은 포인트는 골드 부 와 일반부에 각각 적립한다. 차기 연도부터는 남자 일반부와 골드 부에서 얻은 포인트만 배점한다.
> 2) 여자신인부는 생활체육랭킹대회 및 전국대회에서 우승하였을 경우 우승자(금배, 국화)부와 여자신인부에 각각 그레이드별 랭킹 포인트를 적립한다. 차기 연도부터는 우승자부(금배, 국화)에서 얻은 포인트만 우승자 부(금배, 국화)에 배점한다.

→ 우승 당해년도에는 **한 성적이 두 부서에 동시 적립**된다. 부서별 랭킹은 독립 집계(13-5-4).

### 3.6 유효기간 · 유예 — 승급/강등의 시간 함수

의뢰서에 있던 "전국 최초 동호인 랭킹제, 1위~1000위 연간 산정"은 **규정 원문에서 확인되지 않았다.** 현행 규정이 정하는 것은:

> ⚠️ 선행 문서 `tennis-grade-systems.md` 4.4의 전북 점수제(2/3/5/6/10/12/13점) 서술은 **2026-01-01 개정으로 대체된 것으로 보인다.** 현행 규정에는 해당 점수 체계가 없다.

## 5. 정형화 가능성 판단

### 5.1 결론

**가능 — 단, "포인트 계산"과 "등급 판정"을 분리해야 한다.**

- ✅ **포인트 계산은 완전 정형화 가능.** (대회 그룹 × 진출 라운드) → 점수는 순수 룩업 테이블이다. 4개 협회 모두 동일 구조.
- ⚠️ **랭킹 집계도 정형화 가능.** 베스트N / 전량합산 / 유효기간이 조항으로 명시돼 있다.
- ❌ **등급·부서 판정은 자동화 불가.** 우승 횟수 + 연령 + 선수출신 여부 + 지도자 자격 + 유예 만료가 얽힌 20~40개 분기이고, 다수 조항이 사람 확인(선수등록 이력, 주민등록 연령, 협회 통보)을 전제한다.

### 5.2 제안 스키마

```sql
-- 협회 × 시즌별 규정 스냅샷
create table org_point_rules (
  id             uuid primary key default gen_random_uuid(),
  org_code       text not null,          -- 'gj' | 'jn' | 'kta' | 'kato' | 'jb'
  season_year    int  not null,
  effective_from date not null,          -- 부칙 시행일 (광주 2026-03-17 등)
  source_url     text not null,
  source_revised text,                   -- 문서에 적힌 개정 표기 그대로
  rule_config    jsonb not null,
  unique (org_code, season_year, effective_from)
);
```

`rule_config` 필수 필드:

```jsonc
{
  "point_tables": [                     // 협회마다 2~3개 (우승자/비우승자, OPEN/SEMI/REST)
    {
      "table_key": "winner",            // 광주: winner | non_winner
      "applies_to_divisions": ["coach", "gold", "gukhwa", "geumbae"],
      "rounds": ["champion","runner_up","semi","quarter","r16","r32","r64","entry"],
      "by_grade": {
        "A": {"champion":1000,"runner_up":700,"semi":455,"quarter":182,"r16":60,"r32":30,"entry":5},
        "1": {"champion": 800, "...": 0}
      }
    }
  ],
  "tournament_grades": {                 // 대회 그룹 = 배점 그레이드
    "source": "division_count",          // 광주·KTA·전북 모두 개최 부서 수
    "levels": [ {"code":"A","min_divisions":5}, {"code":"1","min_divisions":4} ],
    "overrides": [ {"match":"광주광역시장배","grade":"A"} ]   // 규정 명시 특례
  },
  "aggregation": {
    "mode": "best_n",                    // best_n | sum_all
    "ranking_n": 25,                     // 광주 제13조 4항
    "award_n": 15,                       // 광주 제27조  ← 값이 다름. 협회 확인 필요
    "reset": "annual",                   // 2026-01-01 리셋(호남 3개권역 공통)
    "expiry": "next_edition_of_same_event"   // 전북. 광주·KTA는 연간
  },
  "eligibility": {
    "min_teams_for_points": 32,          // 광주 13-10, KTA 14-10
    "exempt_divisions": ["coach"],       // 지도자부는 팀수 무관
    "no_point_divisions": ["veteran","couple","non_placer","beginner"],
    "min_round_for_points": "r32"        // KATO 4-2, 전북 9-4
  },
  "cross_org_recognition": {             // 타 단체 우승 인정 여부
    "recognizes": ["kta","kato"],
    "excludes": ["kata","jn"],           // 광주 2026: 전남 제외
    "since": "2026-03-17"

  "grade_validity": {                    // 등급 유효기간 (자동 만료 계산용)
    "placement_to_winner_window_months": 24,   // 광주 제7조 3항
    "promotion_hold_months": 12,
    "winner_grace_years": 3                    // KTA (1회 우승자 한정)
  }
```

### 5.3 자동 계산이 **불가능한** 지점 (반드시 사람/협회 개입)

| # | 지점 | 근거 원문 |
|---|---|---|
| 1 | **대회 그룹 최종 확정** | 광주 제11조 2항 10) "그룹승격은 광주, 전남테니스협회 **최종심의**를 통과하여야 한다" / KATO 3-2-6 "**KATO 정기총회의 최종심의**" / KTA 제12조 7 "랭킹위원회 **최종 심의**" |
| 2 | **그룹 강등 예외 인정** | 광주 제11조 3항 "주경기장의 개·보수 등 **인정할 만한 사유**" → 판단 재량 |
| 3 | **선수출신 / 지도자 판정** | 광주 제17·18조, KTA 제20조 "관할지역 협회의 **레벨 테스트**를 득하여야 한다" |
| 4 | **부서·등급 이동 통보 의무** | 광주 13-5-2) "협회에 **통보**하여 부서와 등급을 조정받아야 하며 통보하지 않아 발생한 착오에 대하여 항변할 수 없다" → **DB가 실제 등급과 어긋날 수 있음이 규정상 전제** |
| 5 | **고의 우승 회피 판정** | 광주 13-14 "**고의적으로** 우승을 회피 할 경우" |
| 6 | **부정선수 사후 포인트 박탈** | 광주 13-15, KATO 6-2 → 확정된 랭킹이 소급 변경될 수 있음 |
| 7 | **명시되지 않은 사항의 유권해석** | 광주 "광주광역시 테니스협회 **유권해석**에 따르며, 동 유권해석의 효력은 명문화된 랭킹 규정과 동일하다" → **규정 텍스트가 완결적이지 않음을 규정 스스로 선언** |
| 8 | **여자 우승자부 페어 추첨** | 광주 제16조 2항 — 대회 현장 추첨. 사전 계산 불가 |

**설계 시사점**: 앱은 랭킹 포인트를 **자체 계산해서 권위 있는 값으로 제시하면 안 된다.** 협회 공표값을 참조·표시하거나, 계산값에는 "참고용, 협회 공표가 우선" 표기가 필요하다. (7번 유권해석 조항 때문에 계산이 맞아도 협회와 달라질 수 있다.)
