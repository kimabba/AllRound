# 올라운드 디자인 베이스라인 — "Active Bold"

> **보관 문서:** 2026-07-18부터 이 방향은 [`pureform-sports-system.md`](pureform-sports-system.md)로 대체되었다.
> 기존 디자인 의사결정의 이력을 위해 삭제하지 않는다.

> 확정: 2026-07-17 (kimabba). 백과장과 **"kimabba가 일단 디자인 변경 → 출시 후 재개선"** 합의됨.
> 이 문서는 **현재 시점의 기본 방향**이며, 계속 개선한다. 실제 반영은 브랜치+PR로만.

## 1. 방향 — Active Bold

에너지틱하되 절제된 스포츠 앱 톤. 크림 베이스 + 볼드 타이포 + 청키 라운드 + 컬러블록 모먼트.
"극강의 깔끔함 × 스포츠". 스켈레톤/다크모드/Pretendard는 기존 인프라 재사용.

## 2. 컬러

| 역할 | 값 | 비고 |
|---|---|---|
| 크림 (베이스) | `#FAF7F2` | surface |
| 잉크 (텍스트/주요 버튼) | `#1A1613` | onSurface, primary action |
| **테니스** | **`#84CC16` (그린)** | 테니스공 색. 주(hero) 종목색 |
| **풋살** | **`#F97316` (오렌지)** | 보조 종목색 |

> ⚠️ **종목색은 기존 코드와 반대로 스왑됨** (이전: 테니스=오렌지/풋살=그린).
>
> **구현 관례 (중요):** 앱 전역이 `sport == 'tennis' ? cs.tertiary : cs.secondary`
> 패턴을 씀. 따라서 ColorScheme에서 **`tertiary` = 테니스 그린 `#84CC16`,
> `secondary` = 풋살 오렌지 `#F97316`**. 종목색은 이 롤 또는 `AppSportColors.forSport()`
> 로만 참조 — 하드코딩 금지(발견 시 토큰으로 교체).

## 3. 타이포

Pretendard Variable. 헤딩은 볼드 헤비(w700~800), 본문 클린. 기존 `AppTypography` 스케일 유지, 히어로/빅넘버만 무게 강조.

## 4. 아이콘 — Phosphor duotone

`phosphor_flutter` (duotone weight). MaterialIcons 대체 방향. 2톤이라 종목색 위에서 깊이가 살아남.
- 네비: 홈/대회/클럽/챗/MY, 활성 = 그린
- 예: trophy, tennis-ball, soccer-ball, map-pin, calendar-blank, users-three, bell, gear-six 등

## 5. 모션 — "카드만 움직인다"

- **공·아이콘은 고정.** 튀거나 떠다니면 이상함. 앰비언트 무한 루프 금지(배터리·산만).
- **카드 섹션만** 스프링으로 살짝: 선택 리프트(~translateY -7, 340ms, press scale .965).
- 로딩=링 스피너 / 성공=체크 스위치 / 칩=첫 진입 스태거 / 토스트=슬라이드+페이드.
- 전부 transform/opacity(GPU). `prefers-reduced-motion` 존중. Flutter는 implicit 위젯 우선.

## 6. 로고 / 앱 아이콘

- 마크: **그린 테니스볼(심 라인) + 오렌지 스윙 아크** = "올라운드"(두 종목 + 모션). 단색 축소 OK, 48px↓ 유지.
- 앱 아이콘: **크림 배경 + 풀컬러 마크**.

## 7. 현재 앱과의 갭 (반영 로드맵)

| 항목 | 현재 | 목표 |
|---|---|---|
| 종목색 | 테니스=오렌지/풋살=그린 | **스왑**(테니스=그린/풋살=오렌지) ← 이 PR |
| 전역 테마 | 블루/슬레이트 M3 (`#1E3A8A`) | 크림 Active Bold (앱 띄워 검증하며 교체) |
| 아이콘 | MaterialIcons | Phosphor duotone |
| 홈 | 없음(랜딩=챗 코치) | 빅넘버+벤토 홈 대시보드 신규 |
| 탭 | 4탭(라운드코치/대회/클럽/더보기) | 재편 검토(홈·대회·클럽·챗·MY) |
| 프로필 | 기존 | 히어로 헤더 + 스탯 + 다크 세그먼트 |

> `app/lib/widgets/app_bottom_nav.dart`(6탭)는 라우터 미사용 죽은 코드 — 정리 대상.

## 8. 반영 순서 (계속 개선)

1. **[이 PR] 종목색 스왑 + 이 문서** — 안전·컴파일 검증.
2. 크림 전역 ColorScheme + Phosphor 네비 — 앱 실행 검증.
3. 프로필 리테마 (히어로 헤더).
4. 홈 대시보드 신규 + 탭 재편.
5. 대회/클럽 화면 점진 마이그레이션.
